# Schematics

**Future feature: Reusable templates for layouts, pages, components, and themes**

## Planned Structure

```
schematics/
├── layouts/          # Layout templates
├── pages/           # Page templates
├── components/      # Component templates
└── themes/          # Theme templates
```

## Purpose

Schematics will allow you to:
- Reuse common layouts across projects
- Share page templates between websites
- Create component libraries
- Define theme variations

## Status

**Not yet implemented** - This is planned for future development.

See `docs/BULK_WEBSITE_GENERATION.md` for details on the planned schematics system.
